# CalSnap

CalSnap is a Python application designed to extract and process calendar-related information from images using advanced text extraction and natural language processing techniques. The application leverages Duckling for time parsing and custom algorithms for text extraction, making it easier for users to manage their scheduling needs.

## Features

- **Image Text Extraction**: Extracts text from images using OCR.
- **Time Parsing**: Utilizes Duckling for accurate natural language date and time parsing.
- **Event Creation**: Allows users to create calendar events based on extracted information.

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- Python 3.x
- pip (Python package installer)

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/JanjanL/CalSnap.git
    ```

2. Navigate to the project directory:

    ```bash
    cd CalSnap
    ```

3. Install the required packages:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Ensure you have an image file named `screenshot.png` in the project directory.
2. Run the main script:

    ```bash
    python calsnap_main.py
    ```

3. Follow the prompts to input the event details.
